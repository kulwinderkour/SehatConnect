# Custom Ringtone for Medicine Reminders

## pill_reminder.mp3

This folder should contain a custom ringtone file named `pill_reminder.mp3`.

### How to add your custom sound:

1. **Find or create an audio file**: 
   - Use a gentle, pleasant sound (not too jarring)
   - Recommended duration: 3-10 seconds
   - Format: MP3 (preferred) or WAV

2. **Place the file here**:
   - Copy your audio file to this directory
   - Name it exactly: `pill_reminder.mp3`

3. **Free sound resources**:
   - freesound.org
   - zapsplat.com
   - soundbible.com

### Example sounds to look for:
- "pill reminder"
- "gentle notification"
- "medical alert"
- "soft chime"

### Technical requirements:
- **Format**: MP3 or WAV
- **File size**: < 500KB recommended
- **Sample rate**: 44.1kHz or 48kHz
- **Bit rate**: 128kbps or higher

---

**Note**: The app is configured to use `pill_reminder.mp3`. If you don't add a custom sound, the system default notification sound will be used instead.
