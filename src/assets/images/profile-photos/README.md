# Profile Photos Directory

This directory contains user profile photos for the SehatConnect app.

## Photo Specifications:
- **Format**: JPG, PNG, or WebP
- **Dimensions**: 200x200 pixels (minimum) or 400x400 pixels (recommended)
- **Aspect Ratio**: 1:1 (square)
- **File Size**: Under 2MB recommended
- **Naming Convention**: Use descriptive names like `rajinder_singh.jpg` or `user_001.png`

## Supported Photo Sizes:
- **Header Profile Photo**: 50x50 pixels (automatically resized)
- **Profile Screen Avatar**: 80x80 pixels (automatically resized)
- **High Resolution**: 200x200+ pixels (for better quality)

## How to Add Photos:
1. Place your photo files in this directory
2. Update the profile image in the app using the profile management system
3. Photos will be automatically resized for different display contexts

## Example Files:
- `rajinder_singh.jpg` (400x400)
- `user_001.png` (200x200)
- `profile_placeholder.jpg` (200x200)
